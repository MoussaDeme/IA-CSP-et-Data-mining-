// Mode d'emploi : 

	La classe Main se situant dans le dossier /src permet d'executer les test de differentes packages.
	Donc il faut compiler le fichier Main.java : javac Main.java.

	* Pour tester le package Representation : 
		 
		-java Main rpt
	
	* Pour tester le backtraking sur l'exemple du file rouge: 

		-java Main ppc

	* Pour tester le planning

		-java Main plan

	* Pour tester le healthcare

		-java Main heal
	

	* Pour tester l'extraction

		-java Main extract
